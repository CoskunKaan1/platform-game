# 🦖 T-Rex Runner: 16-bit x86 Assembly

This project is a text-mode clone of the legendary endless runner game, developed from scratch for the MS-DOS environment using **16-bit x86 Assembly** language. The game is built entirely without any modern game engines or high-level libraries, relying completely on direct hardware port manipulation and BIOS/DOS interrupts.

## ✨ Features

* **Text-Mode Graphics (Video Mode 0003h):** Retro visuals created with standard and extended ASCII characters on an 80x25 terminal screen.
* **Advanced Collision Detection (Hitbox):** Axis-Aligned Bounding Box (AABB) collision algorithm dynamically calculated based on jumping and ducking states.
* **Parallax Background:** A sense of depth created by scrolling clouds, stars, and ground obstacles at different speeds (independent frame timers).
* **Dynamic Difficulty:** Gradual acceleration of the gameplay by progressively decreasing the CPU wait time (delay) as the score increases.
* **Hardware-Level Sound (PC Speaker):** 8-bit jump and crash sound effects generated by directly triggering the `42h`, `43h`, and `61h` hardware ports.
* **Pseudo-Random Number Generation (PRNG):** A custom randomness algorithm utilizing bitwise operations (XOR/AND) on the BIOS timer interrupt (`INT 1Ah`) to spawn cacti and birds at unpredictable intervals.

## 🎮 Controls

| Key | Action |
| :--- | :--- |
| **SPACE** | Jump |
| **S** (or Down Arrow) | Duck (Used to dodge flying birds) |
| **P** | Pause / Resume Game |
| **R** | Restart |
| **ESC** | Exit Game |
| **1, 2, 3** | Difficulty (Speed) selection on the start screen |

## 🛠️ Installation & Execution

This game is a 16-bit MS-DOS application. To run it on modern 64-bit operating systems, you will need an emulator such as [Emu8086].
